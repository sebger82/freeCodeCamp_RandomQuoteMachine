# freeCodeCamp_RandomQuoteMachine
Strona zbudowana na potrzeby kursu freeCodeCamp, pobierajaca cytaty z zewnętrznego serwera i wyświetlająca je losowo użytkownikowi na jego żądanie.

Funkcjonalność w wersji 1.0 z dnia 22.04.2017 r.

1. Pobiera losowy cytat.
2. Wykorzystuje bazę https://quotesondesign.com
3. Pozwala podzielić się nim na tweeterze.
4. Strona w pełni responsywna.
